# 2.5 Accuracy of Position Moves (Display and Measure) - A Axis

## Objective

Evaluate A-axis angular positioning accuracy using:
- **Display:** Galil encoder landing (`_TPA`, `_TEA`)
- **Measure:** external camera + pointer flag angular measurement

## Axis and Instrument Setup

| Item | Value |
|---|---|
| Axis under test | A (free-spinning motor shaft) |
| Encoder | 8192 counts/rev |
| Scale factor | 22.7556 counts/deg |
| 90 deg command | 2048 counts |
| External reference | Pointer flag + camera with center crosshair and X/Y axes |

## Test Script

- Script: `demo_accuracy_A.dmc`
- Recommended gains: `KPA=10`, `KDA=64`, `KIA=0.05`
- Motion profile: `SPA=20000`, `ACA=DCA=200000`
- Settling before read: 2 s

## Procedure

1. Mount pointer flag rigidly on shaft.
2. Align camera center to shaft center.
3. Set physical zero orientation and run script (`DPA=0` at that orientation).
4. Capture snapshots at each landing when prompted.
5. Record:
   - Commanded angle
   - `_TPA`, `_TEA`
   - image ID
   - pointer tip pixel coordinates `(x_tip, y_tip)`

## Post-Processing

Define camera center `(x_ctr, y_ctr)` once per run.

Use:

`theta_meas_deg = atan2((y_tip - y_ctr), (x_tip - x_ctr)) * 180/pi`

Unwrap angles for multi-turn data before comparison.

Compute:

- Encoder angle: `theta_enc_deg = _TPA * 360 / 8192`
- Accuracy error: `e_acc_deg = theta_meas_deg - theta_cmd_deg`
- Display-vs-measure error: `e_dm_deg = theta_meas_deg - theta_enc_deg`

# 2.6 Repeatability of Position Moves (Measure) - A Axis

## Objective

Measure repeatability of returning to the same physical angle after repeated `+1 rev` commands, and assess sensitivity to speed/acceleration.

## Test Script

- Script: `demo_repeatability_A.dmc`
- Gains: `KPA=10`, `KDA=64`, `KIA=0.05`
- Per set: 10 cycles of `PRA=8192` (one full rev each cycle)
- Parameter sets:
  - Set 1: `SPA=8000`, `ACA=DCA=80000`
  - Set 2: `SPA=40000`, `ACA=DCA=400000`
  - Set 3: `SPA=160000`, `ACA=DCA=2000000`

## Procedure

1. Keep same pointer/camera setup as 2.5.
2. Set `DPA=0` at the initial reference orientation.
3. At each cycle landing, capture snapshot + log `_TPA`, `_TEA`.
