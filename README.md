# 486_Application_2

# Project Description
Hardware used:
DMC-4143 motor controller
DPRALTE-020B080 motor driver
AKM21G-ENMN2-00 motor

# Wiring
DMC-4143 connected to DPRALTE-020B080 using MCMD axis C for analog output control
Drive configured in torque mode. Encoder direction reversed in software (CEC=2).
Tuned gains: KP=4.053, KD=500, KI=0
